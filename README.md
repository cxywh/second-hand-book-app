# 二手书交易平台

## 简介
这是一个基于微信小程序的二手书交易平台，用户可以在平台上发布、购买和交易二手书。项目旨在为学生和书籍爱好者提供一个便捷的书籍交易渠道。

## 功能
- **用户登录**：支持微信登录。
- **发布商品**：用户可以发布二手书信息。
- **实时聊天**：买卖双方可以通过平台实时沟通。
- **订单管理**：用户可以查看和管理自己的订单。

## 技术栈
- **前端**：微信小程序原生开发
- **后端**：微信云开发（云函数、云数据库、云存储）
